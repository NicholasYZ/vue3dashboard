# Vue3 Dashboard

一款基于 Vue3+tailwindcss+Element Plus 的后台管理面板

演示地址：<https://v3d.tanxiaoqing.top>

文档正在编写中

### Download

```sh
git clone https://github.com/xiaoqing-tan/vue3dashboard.git
```

### Install
```sh
npm install
```

### Run
```sh
npm run dev
```

### Build
```sh
npm run build
```
